# Post organization

Write posts inside topic folders so the workspace stays easy to scan.

Suggested structure:

```text
_posts/
  hackthebox/
    sherlocks/
      YYYY-MM-DD-title.md
    machines/
      YYYY-MM-DD-title.md
  ctf/
    web/
      YYYY-MM-DD-title.md
    forensics/
      YYYY-MM-DD-title.md
    misc/
      YYYY-MM-DD-title.md
  research/
    web-security/
      YYYY-MM-DD-title.md
    blue-team/
      YYYY-MM-DD-title.md
  blog/
    YYYY-MM-DD-title.md
```

Jekyll still uses the filename date and the post title/permalink, so folders are only for editing organization.
