# Projects & Labs

Selected security labs, reports, CTF results, certificates, and portfolio evidence.

## Selected case studies

### SOC Lab: Windows investigation workflow

- **Focus**: alert triage, host review, evidence collection, and response documentation.
- **Tools**: Windows Event Logs, Sysmon-style thinking, process review, network artifacts.
- **Output**: lab document, setup screenshot, and reusable Blue Team roadmap.
- **Evidence**: [SOC lab document](/assets/portfolio-assets/img/blue/document.pdf), [SOC lab screenshot](/assets/portfolio-assets/img/blue/Lab_SOC/setup1.png), [Blue Team roadmap](/assets/portfolio-assets/img/blue/Course2Roadmap.pdf)

### Ransomware Response Challenge

- **Focus**: suspicious workstation behavior, timeline building, IOC collection, containment steps.
- **Skills shown**: incident response structure, artifact checklist, root-cause thinking.
- **Writeup**: [Ransomware Response Challenge](/posts/ransomware-response-challenge/)

### Beacon Detection Challenge

- **Focus**: repeated outbound traffic, C2-like rhythm, network hunting hypothesis.
- **Skills shown**: traffic grouping, interval analysis, endpoint and network correlation.
- **Writeup**: [Beacon Detection Challenge](/posts/beacon-detection-challenge/)

### Brute-force Windows Login

- **Focus**: repeated failed logons, RDP authentication, successful-login validation.
- **Skills shown**: Windows Event ID mapping, triage flow, post-authentication checks.
- **Writeup**: [Brute-force Windows Login](/posts/brute-force-windows-login/)
