## Đề bài

Đọc file log và in 10 dòng đầu.

## Input mẫu — `input.log`

```text
log line 01
log line 02
log line 03
log line 04
log line 05
log line 06
log line 07
log line 08
log line 09
log line 10
log line 11
log line 12
```

## Kết quả mong đợi

```text
log line 01
log line 02
log line 03
log line 04
log line 05
log line 06
log line 07
log line 08
log line 09
log line 10
```

## Các file

- `main.py`: khung trống để bạn viết lời giải.
- `input.log`: dữ liệu mẫu được `main.py` và `solve.py` đọc tự động.
- `solve.py`: lời giải tham khảo; nên tự làm trước khi mở.
- `expected_output.txt`: kết quả chuẩn dùng để đối chiếu.

## Cách chạy

```powershell
python .\main.py
python .\solve.py
```

Sau khi làm xong, đứng ở thư mục gốc và kiểm tra bài này bằng:

```powershell
python .\check_answer.py 095
```
