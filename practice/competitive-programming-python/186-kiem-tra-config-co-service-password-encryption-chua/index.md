**Nhóm:** H. Regex cho log và config


## Đề bài

Kiểm tra config có service password-encryption chưa.

## Input mẫu — `config.txt`

```text
hostname R1
service password-encryption
enable secret StrongSecret
```

## Kết quả mong đợi

```text
Đã bật
```

## Các file

- `main.py`: khung trống để bạn viết lời giải.
- `config.txt`: dữ liệu mẫu được `main.py` và `solve.py` đọc tự động.
- `solve.py`: lời giải tham khảo; nên tự làm trước khi mở.
- `expected_output.txt`: kết quả chuẩn dùng để đối chiếu.

## Cách chạy

```powershell
python .\main.py
python .\solve.py
```

Sau khi làm xong, đứng ở thư mục gốc và kiểm tra bài này bằng:

```powershell
python .\check_answer.py 186
```
