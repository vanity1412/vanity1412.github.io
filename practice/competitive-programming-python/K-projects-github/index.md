## Cách học

1. Chạy bản demo và đọc dữ liệu đầu vào/đầu ra.
2. Đọc `main.py`, sau đó tự viết lại hoặc tách thành package nhiều module.
3. Hoàn thành checklist nâng cấp trong README từng project.
4. Thêm test, screenshot, license và lịch sử commit trước khi đưa vào CV.
