# CCNA 08 - Inter-VLAN Và Layer 3 Switching

## Tổng Quan

Cấu hình router-on-a-stick, SVI, Layer 3 switching và xử lý lỗi liên VLAN.

Trang này là mục lục của dạng lab. Mỗi bài lab có một trang riêng để dễ quản lý ảnh, cấu hình và ghi chú. Bấm **Đọc bài** sẽ mở trang lab riêng trong tab mới.

## Trọng Tâm Cần Nắm

- Router-on-a-stick
- SVI
- ip routing
- Default gateway theo VLAN
