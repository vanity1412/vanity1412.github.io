# CCNA 10 - Static Và Default Routing

## Tổng Quan

Cấu hình static route, default route IPv4/IPv6 và troubleshoot routing table.

Trang này là mục lục của dạng lab. Mỗi bài lab có một trang riêng để dễ quản lý ảnh, cấu hình và ghi chú. Bấm **Đọc bài** sẽ mở trang lab riêng trong tab mới.

## Trọng Tâm Cần Nắm

- Connected routes
- Static route
- Default route
- Next-hop/exit interface
