# CCNA 11 - OSPF

## Tổng Quan

Triển khai single-area OSPFv2, router ID, neighbor, DR/BDR và default route.

Trang này là mục lục của dạng lab. Mỗi bài lab có một trang riêng để dễ quản lý ảnh, cấu hình và ghi chú. Bấm **Đọc bài** sẽ mở trang lab riêng trong tab mới.

## Trọng Tâm Cần Nắm

- Router ID
- Network command
- Wildcard mask
- Neighbor state
- DR/BDR
