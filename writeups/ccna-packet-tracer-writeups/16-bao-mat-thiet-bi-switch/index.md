# CCNA 16 - Bảo Mật Thiết Bị Và Switch

## Tổng Quan

Cấu hình SSH, password an toàn, port security và hardening cơ bản.

Trang này là mục lục của dạng lab. Mỗi bài lab có một trang riêng để dễ quản lý ảnh, cấu hình và ghi chú. Bấm **Đọc bài** sẽ mở trang lab riêng trong tab mới.

## Trọng Tâm Cần Nắm

- SSH
- Local user
- Password policy
- Port security
- Unused ports
