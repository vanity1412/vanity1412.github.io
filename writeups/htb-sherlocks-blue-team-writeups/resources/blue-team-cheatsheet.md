# Blue Team Cheatsheet

## Report Flow

1. Read the scenario and identify the business context.
2. Inventory evidence and verify hashes when available.
3. Build a timeline before answering final questions.
4. Extract IOCs and validate them against the evidence.
5. Map confirmed behavior to MITRE ATT&CK.
6. Write recommendations that match the observed gaps.

## Common Artifacts

- PCAP files
- Windows Event Logs
- Sysmon logs
- CloudTrail logs
- Web server logs
- Browser history
- Memory images
- Disk images
- Suspicious binaries or scripts

## Useful Tables

- Attack timeline
- IOC list
- Account and host inventory
- MITRE mapping
- Questions and answers
