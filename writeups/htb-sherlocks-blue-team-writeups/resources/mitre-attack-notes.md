# MITRE ATT&CK Notes

| Tactic | Useful Techniques To Check |
| --- | --- |
| Initial Access | Phishing, valid accounts, public-facing applications |
| Execution | PowerShell, command shell, scheduled tasks |
| Persistence | Registry run keys, services, cron, cloud persistence |
| Privilege Escalation | Exploitation, misconfiguration, token abuse |
| Defense Evasion | Obfuscation, log clearing, masquerading |
| Credential Access | Credential dumping, tokens, secrets in files |
| Discovery | Account, system, network, cloud discovery |
| Lateral Movement | SMB, RDP, SSH, remote services |
| Collection | Local files, cloud storage, browser data |
| Exfiltration | Web services, cloud sync, C2 channels |
| Impact | Encryption, data destruction, account lockout |
